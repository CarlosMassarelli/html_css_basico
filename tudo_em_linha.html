Algumas regras para o CSS

É importante preparar um CSS (Cascading Style Sheets) de forma organizada e funcional. Algumas regras são essenciais.

É importante criar um arquivo CSS, para que as alterações e configurações sejam organizadas. Procure nomear os elementos para evitar conflitos de configuração.

A ordem de organização (preferência) dos estilos aplicados ao html é:

    Em ordem de aplicabilidade (verifique no html):

      1. inline, com o atributo "style";
      2. Com a tag "style" <style> e </style>
      3. Com a tag "link" chamando um arquivo externo (.css) com as formatações do html (esta é a forma mais praticada).
      4. Com o elemento "@import" que é utilizado no próprio css, no entanto, está não é uma boa prática.

Como o CSS é aplicado em cascata, precisamos lembrar que as configurações podem ser aplicadas umas sobre as outras, lembrando que a aplicação é feita de cima para baixo... logo, se temos uma formatação no início do arquivo css e fazemos uma nova formatação, do mesmo elemento depois, está será a formatação do elemento.
